## 🚀 البدء في خطوتين (Hot Reload)

### الطريقة المفضلة - تشغيل الكل معاً

```bash
cd /workspaces/AI-Lmstshr
pnpm install
cp .env.example .env.local
pnpm dev
```

✅ سيشغل **Backend** و **Frontend** معاً تلقائياً!
- Backend: http://localhost:3001
- Frontend: http://localhost:3000

---

### أو تشغيل منفصل (إذا أردت)

**Terminal 1 - Backend:**
```bash
cd /workspaces/AI-Lmstshr
pnpm dev:backend
# يعمل على http://localhost:3001
```

**Terminal 2 - Frontend:**
```bash
cd /workspaces/AI-Lmstshr
pnpm dev:frontend
# يعمل على http://localhost:3000
```

### البدء القديم (شامل يدوي)

**خادم:**
```bash
cd artifacts/api-server
pnpm dev
```

**واجهة:**
```bash
cd artifacts/al-mustashar
pnpm dev
```
