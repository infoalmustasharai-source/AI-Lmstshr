🔴 أولاً: تحسين الواجهة (UI/UX) بالكامل
- إعادة تصميم الواجهة بشكل عصري يشبه ChatGPT أو التطبيقات العالمية
- إضافة Sidebar يحتوي على:
  - محادثة جديدة
  - المحادثات السابقة
  - الإعدادات
  - تسجيل الخروج
- إضافة Top Bar يحتوي على:
  - اسم المشروع: "المستشار AI"
  - حالة الاتصال بالذكاء الاصطناعي (Online / Offline)
- تحسين شكل الرسائل:
  - رسائل المستخدم على اليمين
  - ردود AI على اليسار
  - تصميم Bubble احترافي + Shadows + Animations
- استخدام Tailwind + تصميم متجاوب (Mobile / Tablet / Desktop)
- إضافة Animations سلسة (ظهور الرسائل + Hover effects)

🟢 ثانياً: تحسين تجربة المستخدم
- إضافة Loading indicator أثناء انتظار رد AI
- منع إرسال رسالة فارغة
- Scroll تلقائي لآخر رسالة
- زر إعادة إرسال / مسح المحادثة
