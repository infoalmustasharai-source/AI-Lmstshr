## 🐛 استكشاف الأخطاء

### ❌ "خطأ: DATABASE_URL غير معرف"

**الحل:**
```bash
# تأكد من .env.local موجود:
ls -la .env.local

# عامل محتواه:
cat .env.local | grep DATABASE_URL

# إذا كان فارغاً:
cp .env.example .env.local
```

---

### ❌ "خطأ: PORT already in use"

**الحل:**
```bash
# ابحث عن العملية:
lsof -i :3001  # Backend
lsof -i :5173  # Frontend

# أغلقها:
kill -9 <PID>

# أو غير PORT في .env.local
PORT=3002
```

---

### ❌ "خطأ: Cannot find module"

**الحل:**
```bash
# أعد التثبيت:
pnpm install

# أو استخدم الأمر:
pnpm clean
```

---

### ❌ "خطأ: OpenAI response failed"

**الحل:**
```bash
# تحقق من المفتاح:
cat .env.local | grep OPENAI_API_KEY

# إذا كان "sk-temp-free-tier" لا يعمل:
# احصل على مفتاح جديد من: https://platform.openai.com/api-keys

# حدث في .env.local
```
